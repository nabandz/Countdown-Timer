# Countdown-Timer

### :herb: О проекте
Практика по работе с таймерами.

На сайте реализован таймер обратного отсчета. Конечная дата таймера - 1 Января каждого следующего года.

### :herb: Стек технологий
`HTML5` `SCSS` `JavaScript (ES6)` `БЭМ` `Gulp`
